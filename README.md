# oystershell-block-theme

A WordPress block starter theme.

## Theme setup

Find the following text strings (match the case) and replace them with the name of your new theme.

Theme Name: `OysterShell Block Theme`
Text Domain: `oystershell-block-theme`
Function prefix: `oystershell_block_theme_`
@subpackage: `OysterShell_Block_Theme`
